## 규칙
> 충돌 방지 -> **커밋하고 푸시하기 전에 항상 pull**
 
### Folder Tree
```
알고리즘 기초 1
  ㄴ week1
    ㄴ seunzu
      ㄴ 자료구조1
        - 10828 스택.py
        - 9093 단어 뒤집기.py
      ㄴ 자료구조1(연습)
      ㄴ 자료구조1(참고)
  ㄴ week2
    ㄴ seunzu
      ㄴ 수학1
```

### Commit Convention
```
이름 : 백준 번호 이름
ex. seunzu : 자료구조 1 10828 스택
```

### 챌린저스(인증) 규칙
- 7일 중 5일 인증(보증금 5천원)
  - (1번) 백준 문제 성공 인증
    - 하루에 최소 3문제
  - (3번) 리포지토리에 push 인증
